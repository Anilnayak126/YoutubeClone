# YouTube Clone

A YouTube clone built with React and Tailwind CSS, utilizing Rapid API for fetching data.

## Features

- Search for videos
- Watch videos
- Responsive design
- Modern UI with Tailwind CSS

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **API:** Rapid API
